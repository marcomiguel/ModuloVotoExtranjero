ADMINISTRAR PARÁMETROS

En donde pueden administrarse la Fecha de recepción de solicitudes, Fecha de vencimiento de cédula, Estación, Tamaño máximo de imágenes y Visualización de video instructivo.


1. Sección Fecha de recepción de solicitudes

Ésta parte del sistema permite configurar el rango de fechas y horas límite en que se recibirán solicitudes de Traslado Electoral.

Para realizar la configuración se debe de ingresar la fecha y hora Desde, así como la fecha y hora Hasta.

El botón y se habilitarán cuando se ha ingresado al menos un nuevo valor en Desde o Hasta. Una vez realizada la configuración se debe seleccionar el botón o , si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.



2. Sección Fecha de vencimiento de cédula

Ésta parte del sistema permite configurar la fecha máxima de vencimiento de cédula del ciudadano, el cual le permite verificar si un número de cédula se encuentra vigente. Esta configuración se considerará para la pantalla de Solicitud de Traslado Electoral, al momento en que el ciudadano ingresa su número de cédula para realizar el registro.

Para realizar la configuración se debe de ingresar la Fecha.

El botón y se habilitarán cuando se ha ingresado una nueva Fecha. Una vez realizada la configuración se debe seleccionar el botón o , si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.



3. Sección Estación

Ésta parte del sistema permite configurar el N° de Estación, el cual forma parte del N° de Solicitud generado al enviar una solicitud de Traslado Electoral.

Para realizar la configuración se debe de ingresar el N° de Estación.

El botón y se habilitarán cuando se ha ingresado un nuevo N° de Estación. Una vez realizada la configuración se debe seleccionar el botón o , si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.



4. Sección Tamaño de imágenes (en kb)

Ésta parte del sistema permite configurar el tamaño máximo (en kb) que deben tener las imágenes (firma, fotografía y huellas) que adjuntará el ciudadano en la Solicitud de Traslado Electoral.

Para realizar la configuración se debe de ingresar el tamaño correspodiente en kb de las extensiones JPG Y BPM.

El botón y se habilitarán cuando se ha ingresado un nuevo tamaño en la extensión JPG o BPM. Una vez realizada la configuración se debe seleccionar el botón o , si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.



5. Sección Visualización de video instructivo

Ésta parte del sistema permite configurar la visualización y no visualización del video instructivo que se muestra en la Solicitud de Traslado Electoral.

Para realizar la configuración se debe activar o descativar la casilla de verificación asociada a Mostrar video instructivo.

El botón y se habilitarán cuando se haya activado o desactivado la opción de Mostrar Video instructivo. Una vez realizada la configuración se debe seleccionar el botón o , si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.
