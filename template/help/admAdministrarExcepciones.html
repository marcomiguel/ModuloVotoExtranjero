ADMINISTRAR EXCEPCIONES

En esta sección se podrá habilitar o deshabilitar las excepciones del listado, excepciones las cuales se mostrarán en la sección de Recepción cuando un Operador revise el detalle de una Solicitud de Traslado Electoral.


Para realizar la configuración se debe activar o descativar la casilla de verificación de la columna Habilitado, asociada a la respectiva excepción.

El botón se habilitará cuando se haya activado o desactivado al menos una nueva Excepción. Una vez realizada la configuración se debe seleccionar el botón .


Si aparece el mensaje "Se guardaron los cambios correctamente", quiere decir que el sistema registró la configuración realizada.
